## [1.0.0] — Initial Release

### Added

- Custom `MP3CoverReader` class — binary ID3v2.2 / v2.3 tag parser (no external deps)
- Animated vinyl disc with rotating cover art and tone arm
- Pulsing play button with glow ring animation
- Real-time waveform visualizer via Web Audio API
- Album cover blur background with crossfade on track change
- Playlist panel with animated EQ bars for current track
- Session-based Shuffle and Repeat
- Custom seek bar and volume slider
- Keyboard shortcuts: Space, ←/→, S, R, M
- Playback position restore via `localStorage`
- Fully responsive layout (mobile + desktop)
